# EC464_PCB
EC464 - Team 1 - Polar Eyes

The goal of this PCB is to connect different modules together in a clean and concised way. All the though-holes are manually placed using the standard 2.56mm spacing between pins as reference.

For this board I plan to have a total of 4 plate, the first is where all of the traces will be, the second plate will be the universal 5V, the third will be the universal GND, and the last plate is an extra plate to help me route wires.
